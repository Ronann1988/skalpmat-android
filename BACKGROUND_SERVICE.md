# 🌙 Фоновая служба — Работа при заблокированном экране

## 📋 Что делает фоновая служба

Когда вы включаете переключатель **"🌙 Работать в фоне"**:

1. **Торговля продолжается** даже когда:
   - Экран телефона выключен
   - Телефон заблокирован
   - Приложение свёрнуто
   - Вы используете другие приложения

2. **Уведомление в статус баре** показывает:
   - 🟢 Статус: "Бот торгует..."
   - 💰 Текущий PnL
   - 📊 Активные позиции

3. **Android не убивает приложение** — система видит активную службу и не закрывает его для экономии памяти

---

## 🎯 Как использовать

### Включение:

1. Откройте приложение
2. **Перед запуском бота** включите переключатель:
   ```
   🌙 Работать в фоне (при заблокированном экране) [✓]
   ```
3. Нажмите **"▶️ Старт"**
4. Теперь можно блокировать телефон — бот продолжит работу

### Выключение:

1. Откройте приложение
2. Выключите переключатель
3. Нажмите **"⏹️ Стоп"**

### Когда служба активна:


## ⚙️ Технические детали

### Разрешения Android:

```xml
<uses-permission android:name="android.permission.FOREGROUND_SERVICE" />
<uses-permission android:name="android.permission.WAKE_LOCK" />
<uses-permission android:name="android.permission.ACCESS_NETWORK_STATE" />
<uses-permission android:name="android.permission.INTERNET" />
```

### Тип службы:

- **Foreground Service** — видимая служба с уведомлением
- **Приоритет**: LOW (не беспокоит пользователя)
- **Канал уведомлений**: `skalpmat_service_channel`

## 🔋 Влияние на батарею

Фоновая служба **минимально влияет** на батарею:

- ✅ Бот не использует GPS постоянно
- ✅ Экран не включается
- ✅ Сеть используется только для API запросов (раз в 15 сек)
- ⚠️ Основное потребление: поддержание сетевого соединения

**Ожидаемое потребление**: ~2-5% в час (зависит от частоты сканирования)

## 🛡️ Безопасность

### Почему нужно уведомление?

Android (начиная с версии 8.0) требует показывать уведомление для всех фоновых служб. Это:

1. **Защита пользователя** — вы всегда видите, что работает в фоне
2. **Контроль** — можно быстро остановить службу
3. **Прозрачность** — видно какое приложение использует ресурсы

### Можно ли отключить уведомление?

❌ **Нет**, Android заблокирует службу без уведомления.

Но можно:
- Скрыть с lock screen (настройки телефона)
- Отключить звук/вибрацию (уже сделано)

## ❓ FAQ

### Q: Уведомление мешает, можно скрыть?
**A:** Да, в настройках телефона:
1. Настройки → Приложения → SKALPMAT V7
2. Уведомления → Канал "Trading Service"
3. Отключите "На экране блокировки"

### Q: Служба разряжает батарею?
**A:** Минимально (~2-5%/час). Основное потребление — сетевые запросы к Gate.io.

### Q: Что будет если закрыть приложение?
**A:** Служба продолжит работать. Уведомление останется в статус баре.

### Q: Как полностью остановить?
**A:** 
1. Откройте приложение
2. Выключите переключатель "Работать в фоне"
3. Нажмите "Стоп"

### Q: Можно сделать без уведомления?
**A:** Нет, Android запрещает скрытые фоновые службы (policy restriction).

### Q: Работает ли на iOS?
**A:** Нет, эта функция только для Android. iOS имеет другие ограничения.

## 🐛 Troubleshooting

### Служба не запускается

**Проверьте:**
1. Android 8.0+ (требуется для каналов уведомлений)
2. Разрешения выданы
3. Уведомления не заблокированы в настройках

### Уведомление не показывается

**Решение:**
```
Настройки → Приложения → SKALPMAT V7 → Уведомления
→ Включите "Trading Service"
```

### Бот останавливается через 10 минут

**Причина:** Android убивает процесс для экономии памяти.

**Решение:**
1. Настройки → Батарея → SKALPMAT V7
2. Выберите "Не ограничивать"
3. Отключите оптимизацию батареи

### Телефон сильно греется

**Причина:** Частые сетевые запросы.

**Решение:**
- Увеличьте `SCAN_INTERVAL` в `.env` (например, до 30 сек)
- Уменьшите количество пар для сканирования
